## 🎯 Objective
The objective of this assignment is to perform sentiment analysis on tweets and classify them into positive, neutral, and negative categories using machine learning models.

---

## 📌 Topic Chosen
Shah Rukh Khan Tweets

This topic was selected because Shah Rukh Khan is a globally popular celebrity and generates diverse opinions on social media.

---

## 📂 Dataset
- 100 tweets collected manually  
- Each tweet labeled as:
  - Positive  
  - Neutral  
  - Negative  

---

## ⚙️ Preprocessing Steps
- Converted text to lowercase  
- Removed unnecessary characters  
- Cleaned text data  

---

## 🔀 Data Splitting
- 80% training data (80 tweets)  
- 20% testing data (20 tweets)  

---

## 🤖 Models Used
- Naïve Bayes  
- Logistic Regression  
- Support Vector Machine (SVM)  

---

## 📊 Results

| Model | Precision | Recall |
|------|----------|--------|
| Naïve Bayes | 0.73 | 0.65 |
| Logistic Regression | 0.60 | 0.60 |
| SVM | 0.56 | 0.55 |

---

## 🧠 Conclusion
Naïve Bayes classifier performed best with highest precision and recall compared to other models.
